# cmuxser 
> 版本 `@cmuxser` 1.0.0

## 介绍：
>基于node构建服务端项目
>基于koa实现mvc项目管理
>配套多种模板（ts,node-mysql,node-mongo等）
>配套多种部署机制（docker,serverless）



## 下载安装

检查是否安装过cmuxser

```bash
    cmuxser -V
```
### 初次安装
```bash
    npm install -g cmuxser
    yarn global add cmuxser
```

## 创建项目：

```bash
    cmuxser init
```
